# Пошаговая инструкция

## 1. Получение Perplexity API ключа

### Регистрация:
1. Откройте https://www.perplexity.ai/settings/api
2. Войдите или зарегистрируйтесь
3. Нажмите **"Generate API Key"**
4. Скопируйте ключ (начинается с `pplx-`)

### Стоимость:
- **Первые $5 бесплатно** при регистрации
- `sonar-pro`: $1 за 1M токенов (~500-1000 запросов)
- **1 SMS запрос ≈ $0.001** (меньше рубля)

## 2. Сборка приложения

### Скачайте проект:
```bash
git clone https://github.com/neizvedaniydag/sms-ai-gateway.git
cd sms-ai-gateway
```

### Установите Android Studio:
- Скачайте: https://developer.android.com/studio
- Установите Android SDK (API 34)

### Соберите APK:
1. Откройте проект в Android Studio:
   - `File` → `Open` → выберите папку `sms-ai-gateway`
   
2. Дождитесь синхронизации Gradle (5-10 мин)

3. Соберите APK:
   - `Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
   
4. APK создастся здесь:
   ```
   app/build/outputs/apk/debug/app-debug.apk
   ```

## 3. Установка на телефон

### Перенос APK:
```bash
# Через ADB
adb install app/build/outputs/apk/debug/app-debug.apk

# Или просто скопируйте на телефон и откройте
```

### Первый запуск:
1. Откройте **SMS AI Gateway**
2. Разрешите:
   - ✅ Получение SMS
   - ✅ Отправка SMS
   - ✅ Уведомления
   
3. Вставьте **Perplexity API ключ**
4. Нажмите **Сохранить**

## 4. Оптимизация работы

### Отключите энергосбережение:
1. Настройки → Приложения
2. SMS AI Gateway
3. Батарея → **Без ограничений**

### Отключите автозасыпание (на время контрольной):
1. Настройки → Экран
2. Тайм-аут экрана → **30 минут** (или никогда)

### Подключение к интернету:
- Wi-Fi (рекомендуется) или мобильный интернет
- Убедитесь что сигнал стабилен

## 5. Использование

### Просто отправьте SMS:
```
Ваш телефон (основной)
    ↓ SMS: "Какой радиус Земли?"
Второй телефон (с приложением)
    ↓ Поиск в Perplexity (5-10с)
    ↓ SMS обратно: "Радиус Земли 6371 км"
Ваш телефон ← Получаете ответ
```

### Примеры запросов:
- "Что такое квантовая запутанность?"
- "GPT-4 вышел в 2023 или 2024?"
- "Сколько людей в Москве?"

## 6. Отладка

### Просмотр логов:
```bash
adb logcat | grep SmsAiGateway
```

### Частые проблемы:

**SMS не приходит ответ:**
- Проверьте API ключ (должен быть `pplx-...`)
- Проверьте баланс API: https://www.perplexity.ai/settings/api
- Убедитесь что есть интернет (Wi-Fi или мобильный)

**Приложение не получает SMS:**
- Проверьте разрешения (RECEIVE_SMS, SEND_SMS)
- Убедитесь что приложение не в энергосбережении

**"Ошибка: API error 401":**
- Неправильный API ключ - пересоздайте на perplexity.ai

**"Ошибка: API error 429":**
- Исчерпан лимит запросов - подождите 1 минуту

## 7. Настройка модели

Если хотите изменить модель:

Редактируйте `app/src/main/java/com/aigateway/sms/AiSmsService.kt`:

```kotlin
// Строка ~90
put("model", "sonar-pro")  // Или "sonar" для экономии
```

Модели:
- `sonar-pro` - Веб + академические источники ($1/1M)
- `sonar` - Только веб-поиск ($0.2/1M)
