# SMS AI Gateway 🚀

Android-приложение для автоматической пересылки SMS в Perplexity AI с веб-поиском и академическими источниками. Ответ отправляется обратно SMS.

## Как это работает

```
Входящая SMS → Проверка белого списка → Perplexity Sonar API → Ответ SMS
```

## Возможности

- ✅ **Автоматическая обработка SMS** в фоновом режиме
- ✅ **🔒 Белый список номеров** - только доверенные люди получают ответы
- ✅ **Perplexity Sonar Pro** - поиск в веб + академических источниках
- ✅ **Надёжный ForegroundService** - не убивается Android
- ✅ **Точная адресация** - ответ уходит именно тому, кто спросил
- ✅ **Логирование запросов** для отладки

## 🔒 Безопасность: Белый список

### Как работает

Приложение **отвечает только номерам из белого списка**. Это защищает от:

- ❌ **Спама** - посторонние не получат ответы
- ❌ **Утечки информации** - API ключ работает только для вас
- ❌ **Потери SMS** - ответы не уходят куда-то не туда
- ❌ **Нежелательных запросов** - дети/коллеги не используют ваш API

### Настройка белого списка

1. Откройте приложение на втором телефоне
2. В поле **"Белый список"** введите номера доверенных людей:

```
+79991234567
+79857654321
89123456789
```

3. Нажмите **"💾 Сохранить настройки"**

### Форматы номеров

Приложение умно сравнивает номера - любой формат совпадёт:

```
+7 (999) 123-45-67
+79991234567
89991234567
79991234567
9991234567
```

Все эти варианты распознаются как **один номер**.

### Пустой список = Всем разрешено

⚠️ Если белый список **пустой**, приложение отвечает **ВСЕМ**. Это удобно для тестирования, но небезопасно для продукшена.

**Рекомендация:** Добавьте хотя бы 1 номер для защиты.

### Гарантия точной адресации

✅ **Ответ всегда уходит тому, кто спросил:**

```
Ваш телефон (+79991234567) → SMS: "Какой радиус Земли?"
                                    ↓
                     Второй телефон (проверяет белый список)
                                    ↓
                              Perplexity API
                                    ↓
Ваш телефон (+79991234567) ← SMS: "Радиус Земли 6371 км"
```

**Никакой путаницы!** Каждый ответ идёт точно на номер отправителя.

## Быстрый старт

### 1. Получите API ключ Perplexity

1. Зарегистрируйтесь на [perplexity.ai/settings/api](https://www.perplexity.ai/settings/api)
2. Скопируйте API ключ
3. Первые $5 бесплатно

### 2. Сборка приложения

```bash
git clone https://github.com/neizvedaniydag/sms-ai-gateway.git
cd sms-ai-gateway
```

Откройте проект в **Android Studio**:

1. File → Open → выберите папку `sms-ai-gateway`
2. Build → Build Bundle(s) / APK(s) → Build APK(s)
3. APK сохранится в `app/build/outputs/apk/debug/app-debug.apk`

### 3. Установка и настройка

1. Установите APK на **второй телефон** (который будет работать шлюзом)
2. При первом запуске:
   - Введите **Perplexity API ключ**
   - Добавьте номера в **белый список** (каждый с новой строки)
   - Разрешите доступ к SMS и уведомлениям
3. Оставьте телефон подключённым к Wi-Fi

### 4. Использование

Просто отправьте SMS с **номера из белого списка** на номер второго телефона. Приложение:

1. Проверит номер в белом списке ✅
2. Отправит вопрос в Perplexity (с веб-поиском)
3. Автоматически ответит SMS **именно вам** с результатом

## Настройки

### Файл конфигурации

API ключ и белый список хранятся в SharedPreferences. Изменить можно:

- В самом приложении (главный экран)
- Или в коде: `app/src/main/java/com/aigateway/sms/Config.kt`

### Модели Perplexity

| Модель | Описание | Цена |
|--------|----------|------|
| `sonar-pro` | Веб + академический поиск | $1/1M токенов |
| `sonar` | Только веб-поиск | $0.2/1M токенов |

Изменить модель: `AiSmsService.kt` строка 90

## Структура проекта

```
app/src/main/java/com/aigateway/sms/
├── SmsReceiver.kt          # BroadcastReceiver для приёма SMS + проверка белого списка
├── AiSmsService.kt         # ForegroundService для обработки
├── MainActivity.kt         # Главный экран настроек
├── SmsSender.kt            # Отправка SMS
└── Config.kt               # Конфигурация + белый список
```

## Требования

- Android 8.0+ (API 26+)
- Разрешения: RECEIVE_SMS, SEND_SMS, INTERNET, FOREGROUND_SERVICE
- Wi-Fi или мобильный интернет
- Perplexity API ключ

```bash
adb logcat | grep SmsAiGateway
```
